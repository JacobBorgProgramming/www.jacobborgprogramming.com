Fruity Web Browser

Fruity is a minimalist web browser designed for shared devices.

The Fruity Web Browser is designed to run smoothly on computers running MacOS 10.12 or newer. Fruity features a "Delete all" button to clear the browser and limited tracking. No personal data is ever recorded.
