JazzBlocks CSS and JS Libraries

I started developing JazzBlocks a couple years ago when I became annoyed with how much third-party development sources (like CSS libraries) tracked users.

JazzBlocks is 100% free and I am currently working on the next generation of it.

You can find it here.

Atomic Password Generator

I began developing the Atomic Password Generator because I hate creating passwords. Atomic doesn't have privacy-harming tracking or data recording and the passwords created by it I can never access.

Atomic Password Generator is 100% free.

You can find it here.
